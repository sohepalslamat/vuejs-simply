# V-Model in Vue2 and Vue3

بعد أن تنسخ المستودع إلى حاسبك بشكل محلي تقوم بالتالي:

------------------- Vue2 ------------------
## الخطوة الأولى 
```
cd vue-2
```
## تنصيب مكتبات المشروع
```
npm install
```

### تشغيل السيرفر بوضع التطوير
```
npm run serve
```
------------------------------------------
------------------- Vue3 -----------------
## الخطوة الأولى 
```
cd vue-3
```
## تنصيب مكتبات المشروع
```
npm install
```

### تشغيل السيرفر بوضع التطوير
```
npm run dev
```
